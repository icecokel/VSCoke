# CSS Animation 속성

**@keyframes (키프레임)**:

- 애니메이션의 각 단계를 정의하는 키프레임 규칙을 생성합니다. 이 규칙은 'from' (시작 상태) 및 'to' (끝 상태) 키워드 또는 특정 백분율로 지정할 수 있습니다.

### 추가 설명

1. **`animation-timing-function`:**

   1. **ease**:
      - **`ease`** 타이밍 함수는 가장 일반적으로 사용되며, 처음과 끝에서는 천천히 시작하고 중간에 빠르게 진행되는 애니메이션을 만듭니다.
      - 이것은 자연스러운 움직임을 제공하며 대부분의 애니메이션에 적합합니다.
   2. **linear**:
      - **`linear`** 타이밍 함수는 애니메이션의 진행이 일정한 속도로 진행됩니다.
      - 처음부터 끝까지 일정한 속도로 움직이므로 자연스럽지 않을 수 있으며, 주로 일반적인 애니메이션에서는 사용 되지 않습니다. 하지만 특별한 경우에는 유용할 수 있습니다.
   3. **ease-in**:
      - **`ease-in`** 타이밍 함수는 애니메이션이 천천히 시작하여 점점 빨라지는 애니메이션을 만듭니다.
      - 처음에는 느리게 시작하기 때문에 부드럽게 가속하는 효과를 제공합니다. 주로 요소가 화면에 나타나는 효과에 사용됩니다.
   4. **ease-out**:
      - **`ease-out`** 타이밍 함수는 애니메이션이 빠르게 시작하여 점점 느려지는 애니메이션을 만듭니다.
      - 처음에는 빠르게 시작하여 끝부분에서는 느리게 감속하므로 요소가 화면에서 사라지는 효과에 적합합니다.
   5. **ease-in-out**:
      - **`ease-in-out`** 타이밍 함수는 애니메이션이 천천히 시작하여 중간에 가속한 후, 끝에서는 다시 느려지는 애니메이션을 만듭니다.
      - 애니메이션의 시작과 끝에서는 느리게 움직이며, 중간에는 빠르게 움직입니다. 이로써 부드러운 애니메이션 효과를 제공합니다.

2. **`animation-iteration-count`**:
   - 값으로 숫자를 사용하며, **`infinite`**를 사용하여 무한 반복할 수 있습니다.
   - 예시: **`animation-iteration-count: 3;`**는 애니메이션을 3번 반복하도록 설정합니다.
3. **`animation-direction`**:
   - 종류: **`normal`** (정방향), **`reverse`** (역방향), **`alternate`** (정방향과 역방향을 번갈아가며 반복), **`alternate-reverse`** (역방향과 정방향을 번갈아가며 반복)가 있습니다.
4. **`animation-fill-mode`**:
   1. **none**:
      - **`animation-fill-mode: none;`**으로 설정하면 애니메이션의 재생이 시작하기 전과 끝난 후에 요소에 어떤 스타일도 적용되지 않습니다.
      - 애니메이션 이전과 이후의 스타일은 해당 스타일 규칙에 따라 변하지 않습니다.
   2. **forwards**:
      - **`animation-fill-mode: forwards;`**로 설정하면 애니메이션이 끝난 후에는 요소에 애니메이션의 마지막 키프레임의 스타일이 유지됩니다.
      - 애니메이션 이전에 적용된 스타일은 유지되지 않고, 애니메이션 이후의 스타일만 유지됩니다.
   3. **backwards**:
      - **`animation-fill-mode: backwards;`**로 설정하면 애니메이션이 시작하기 전에 애니메이션의 첫 번째 키프레임의 스타일이 요소에 적용됩니다.
      - 애니메이션 이후에는 해당 스타일이 유지되지 않으며, 애니메이션 이전의 스타일로 돌아갑니다.
   4. **both**:
      - **`animation-fill-mode: both;`**로 설정하면 애니메이션이 시작하기 전에는 첫 번째 키프레임의 스타일이 적용되고, 애니메이션 이후에는 마지막 키프레임의 스타일이 유지됩니다.
      - 즉, 애니메이션 이전과 이후의 두 가지 스타일이 요소에 적용됩니다.
5. **`animation-play-state`**:
   - **`animation-play-state`** 속성은 애니메이션의 재생 상태를 제어합니다.
   - 종류: **`running`** (애니메이션 재생), **`paused`** (애니메이션 일시 중지).
   - 예시: **`animation-play-state: paused;`**는 애니메이션을 일시 중지합니다.

- **`animation-play-state`** 예시

  ```jsx
  <!DOCTYPE html>
  <html lang="en">
    <head>
      <meta charset="UTF-8" />
      <meta name="viewport" content="width=device-width, initial-scale=1.0" />
      <title>Animation Play State 예제</title>
      <style>
        .box {
          width: 100px;
          height: 100px;
          background-color: blue;
          animation-name: moveBox;
          animation-duration: 2s;
          animation-timing-function: linear;
          animation-iteration-count: infinite;
        }

        @keyframes moveBox {
          from {
            transform: translateX(0);
          }
          to {
            transform: translateX(200px);
          }
        }

        button {
          margin-top: 20px;
        }
      </style>
    </head>
    <body>
      <div class="box"></div>
      <button onclick="pauseAnimation()">일시 정지</button>
      <button onclick="resumeAnimation()">재개</button>

      <script>
        const box = document.querySelector('.box');
        const animation = box.style.animation;

        function pauseAnimation() {
          box.style.animationPlayState = 'paused';
        }

        function resumeAnimation() {
          box.style.animationPlayState = 'running';
        }
      </script>
    </body>
  </html>
  ```
