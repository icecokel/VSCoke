## 개요 (Overview)

의료·임상 분석 플랫폼에서 복잡한 시뮬레이션 입력, 서버 데이터 연동, 결과 리포트, 인증·세션, 배포 전 검증 흐름을 프론트엔드 제품 기준으로 정리했습니다.

## 기술 스택

Next.js · React · TypeScript · TanStack Query · Zustand · React Hook Form · Playwright · GitHub Actions · Docker

## 주요 작업

- 환자 정보, 질병, 약물, 검사값, 분석 조건이 연결되는 입력 흐름을 사용자 진행 단계와 서버 요청 데이터 기준으로 구조화해 입력-검증-실행-결과 확인 흐름을 개발했습니다.
- 서버 기능 명세 변경과 조건별 예외로 화면 상태와 요청 데이터가 어긋날 수 있는 문제를 요청·응답 타입, 실행 조건, 저장값·엑셀 데이터 복원 흐름 정리로 보정했습니다.
- 분석 결과를 테이블, 차트 확대 보기, 문서 미리보기·다운로드, 엑셀 불러오기·내보내기로 확인하고 공유할 수 있도록 결과 활용 흐름을 구성했습니다.
- 여러 프론트엔드 제품의 로그인, 세션 만료, 제품 이동, 모바일 접근 정책을 공통 진입점과 세션 만료 경험 기준으로 정리했습니다.
- 프로젝트별 실행 환경, 의존성, 패치 버전, 타입 검사, 빌드 검증, 브라우저 자동 테스트, 배포 스크립트를 정리해 배포 전 검증 기준을 보강했습니다.

## 성과 (Outcome)

복잡한 의료·임상 분석 업무를 사용자가 입력, 실행, 결과 확인, 문서·엑셀 출력까지 이어갈 수 있는 프론트엔드 흐름으로 정리하고, 인증·세션과 배포 전 검증 기준을 함께 보강했습니다.
