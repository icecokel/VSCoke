## 기본 정보

- 회사: 주식회사 코드크레용
- 기간: 2024.09 - 2025.04
- 고용 형태: 정규직
- 핵심 포지션: 모바일 웹뷰 기반 Playground 기획·구축
- 기술 스택: Next.js, React, TypeScript, Phaser, Tailwind CSS, shadcn/ui, Flutter bridge, Airbridge

## 핵심 요약

앱 안에서 콘텐츠를 빠르게 실험하고 업데이트할 수 있도록 Playground를 기획·구축했습니다.

## 문제

- 앱 콘텐츠를 빠르게 실험하려면 네이티브 배포 주기에 묶이지 않는 웹뷰 기반 제품 흐름이 필요했습니다.
- 모바일 웹뷰 환경에서는 키보드, safe area, 로딩, 앱과 웹 간 통신, 게임 렌더링 제약을 함께 다뤄야 했습니다.

## 해결·결과

- 이벤트·게임형 콘텐츠를 Next.js 기반 모바일 웹뷰로 구현했습니다.
- Phaser 기반 게임, 랭킹·결과 제출, 배너·사운드 제어, Flutter bridge, 딥링크·마케팅 SDK 연동을 구현했습니다.
- 앱 배포와 독립적으로 콘텐츠를 개선할 수 있는 웹뷰 기반 실험 공간을 만들었습니다.
