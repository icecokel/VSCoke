## 기본 정보

- 회사: 주식회사 코드크레용
- 기간: 2025.02 - 2025.07
- 고용 형태: 정규직
- 핵심 포지션: 자막 번역 관리 도구로 콘텐츠 운영 자동화
- 기술 스택: TypeScript, AWS Bedrock, OpenAI API, Google Cloud Translate, Express

## 핵심 요약

CLI로 처리하던 자막 번역을 웹 관리 도구로 옮겨 운영자가 직접 처리할 수 있게 했습니다.

## 문제

- 자막 번역 작업이 CLI와 개발자 중심 흐름에 머물러 운영자가 직접 다루기 어려웠습니다.
- 콘텐츠 공급 속도를 높이려면 자막 파일 처리, 언어별 프롬프트, 번역 검수, 캐릭터 매칭을 관리 화면에서 처리할 필요가 있었습니다.

## 해결·결과

- 자막 파일 처리, 언어별 프롬프트, 번역 검수, 캐릭터 매칭을 관리 화면에서 처리하게 했습니다.
- 반복 번역 업무의 팀 리소스를 90% 이상 줄이고, 작업 생산성을 10배 이상 높였습니다.
- 운영자가 하루 6~8개 작품을 처리할 수 있게 했습니다.
