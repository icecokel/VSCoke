## 개요 (Overview)

임상시험 데이터를 검색하고, 조건을 설정해 분석 결과를 확인하는 화면을 개발했습니다.

## 기술 스택

Next.js · React · TypeScript · TanStack Query · Zustand · Material-UI · ECharts

## 주요 작업

- 검색 조건, 결과 대시보드, 분석 설정 화면의 데이터 흐름을 개선했습니다.
- 여러 약물과 분석 조건을 선택하는 복잡한 입력 상태를 보정했습니다.
- 저장된 분석 설정을 다시 불러오는 과정에서 발생하는 예외를 처리했습니다.
- 결과 차트 확대 보기, 리포트 배치, 모달, 날짜 선택 등 화면 사용성을 개선했습니다.
