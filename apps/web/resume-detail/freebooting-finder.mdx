## 기본 정보

- 회사: 주식회사 코드크레용
- 기간: 2025.06 - 2025.06
- 고용 형태: 정규직
- 핵심 포지션: 무단 도용 콘텐츠 검색 도구
- 기술 스택: TypeScript, Express, Axios, Cheerio, Vercel Serverless Functions

## 핵심 요약

반복 수동 검색에 의존하던 무단 도용 콘텐츠 확인 업무를 크롤링 기반 도구로 옮겼습니다.

## 문제

- 콘텐츠 무단 도용 여부를 매일 수동으로 확인해야 해 운영 시간이 반복적으로 소모됐습니다.
- 브라우저 직접 크롤링에서 봇 차단을 겪는 구간이 있어 확인 업무를 안정적으로 분리할 필요가 있었습니다.

## 해결·결과

- 검색 키워드 기반 크롤링 도구와 결과 확인 화면을 만들었습니다.
- 매일 50분 걸리던 확인 업무를 10분 이내로 줄였습니다.
- 브라우저 직접 크롤링에서 봇 차단을 겪는 구간은 Node.js 기반 처리로 분리했습니다.
