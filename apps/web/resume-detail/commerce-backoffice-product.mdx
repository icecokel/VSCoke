## 기본 정보

- 회사: 주식회사 코드크레용
- 기간: 2023.05 - 2025.07
- 고용 형태: 정규직
- 핵심 포지션: 초기 멤버로 커머스·백오피스 제품 개발
- 기술 스택: Next.js, React, TypeScript, MUI, Tailwind CSS, Jotai, SWR, React Hook Form

## 핵심 요약

초기 멤버로 합류해 고객용 이커머스와 운영자용 백오피스를 함께 개발했습니다.

## 문제

- 고객용 서비스와 운영자 업무 화면을 동시에 고도화해야 했고, 기존 백오피스는 운영자 사용성과 유지보수성을 개선할 필요가 있었습니다.
- 주문, 상품, 셀러, 정산, 이벤트, 고객 문의 등 운영 데이터가 흩어져 반복 작업 비용이 컸습니다.

## 해결·결과

- 셀렉터스 고객용 웹에서 다국어, 결제, 선물하기, 드로우, 상품·주문·마이페이지 흐름을 개발했습니다.
- Flutter Web 기반 백오피스를 Next.js로 마이그레이션하고, 주문·상품·셀러·정산·이벤트·문의 관리 화면을 구축했습니다.
- 검색 조건 초기화, CSV·Excel export, 드래그 앤 드롭 정렬, 공통 테이블·다이얼로그·폼 컴포넌트를 적용해 운영자 반복 작업을 줄였습니다.
