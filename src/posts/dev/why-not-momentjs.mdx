### **3\. 사용법 예시**

**Format**

```
const dateFromMoment = moment().format("YYYY/MM/DD") // "2023/03/06"

const dateFromLuxon = DateTime.now().toFormat("yyyy/LL/dd") // "2023/03/06"
```

아마 format()을 쓰시는 분들이 적지 않을 것이라고 생각합니다. 주의 하실 점은, 년 월 일을 표시하는 알파벳이 바뀐점인데요. 처음엔 조금 불편하겠지만, 이 부분은 익숙해지기 쉬운 쪽이라고 생각합니다.

**timezone**

이전에 저와같이 타임존이 중요하신 분들은 "mement-timezon"을 추가적으로 설지하셔서 사용하셨을 것이라고 생각합니다. luxon에서는 별도 설치 없이 timezone 설정이 가능합니다 

```
const nowFromMoment = moment().tz("America/New_York").fromat();
const nowFromLuxon = DataTime.now().set({millisecond : 0}).setZone("America/New_York").toISO({suppressMilliseconds : true});
```

이전보다 길어지긴 했지만, 추가 라이브러리 설치하지 않아도 되는것이 좋습니다.

중간에 millisecond를 0으로 바꾸고, toISO에 설정값을 준 이유는, 기존 moment-timezone을 이용해서 단순 format만 했을때, millsecond는 생략되어 나왔기 때문에 포맷을 맞추기 위함입니다. millisecond가 offset에 포함되어도 괜찮으신 분들은 생략해도 좋습니다.
