# 블로그를 시작합니다

안녕하세요! 이 블로그는 **Next.js**와 **MDX**를 활용하여 만들었습니다.

## 기술 스택

이 블로그는 다음 기술들을 사용합니다:

- **Next.js 15** - React 프레임워크
- **MDX** - Markdown + JSX
- **Tailwind CSS** - 스타일링
- **Shiki** - 코드 하이라이팅

## 코드 예시

아래는 TypeScript 코드 예시입니다:

```typescript
const greeting = (name: string): string => {
  return `Hello, ${name}!`;
};

console.log(greeting("World"));
```
