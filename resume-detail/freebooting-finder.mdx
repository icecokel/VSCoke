## 개요 (Overview)

**Freebooting Finder**는 Dailymotion 등의 플랫폼에서 영상 무단 도용을 탐지하기 위한 크롤러 서비스입니다. Vercel Serverless 환경에서 동작하도록 설계되었으며, 웹 크롤링을 통해 데이터를 수집합니다.

## 주요 기능 (Key Features)

- **무단 도용 콘텐츠 탐지**: Dailymotion 등에서 특정 키워드나 패턴을 기반으로 무단 업로드된 영상을 식별합니다.
- **웹 크롤러 구현**: `cheerio`와 `axios`를 활용하여 대상 사이트의 HTML을 파싱하고 필요한 메타데이터를 추출합니다.
- **API 서비스**: 수집된 데이터를 클라이언트에서 조회할 수 있도록 REST API 엔드포인트를 제공합니다.

## 기술적 도전 (Technical Challenges)

- **Serverless 환경 최적화**: Vercel 배포를 위해 기존 Express 서버 구조를 Serverless Function(`api/index.ts`) 형태로 리팩토링했습니다.
- **동적 Base URL**: 배포 환경에 따라 유동적으로 변하는 URL을 처리하기 위해 클라이언트 로직을 개선했습니다.
