## 개요 (Overview)

임상시험 데이터를 검색하고, 조건을 설정해 분석 결과를 확인하는 화면을 개발했습니다.

## 기술 스택

Next.js · React · TypeScript · TanStack Query · Zustand · Material-UI · ECharts

## 주요 작업

- 검색 조건, 결과 대시보드, 분석 설정 화면 사이에서 데이터가 이어지지 않는 문제를 화면 흐름 기준으로 보정했습니다.
- 여러 약물과 분석 조건을 선택하는 과정에서 입력 상태가 복잡해지는 문제를 상태 관리 기준으로 정리했습니다.
- 저장된 분석 설정을 다시 불러오는 과정에서 발생하는 예외를 처리해 분석 조건 복원 흐름을 개선했습니다.
- 결과 차트 확대 보기, 리포트 배치, 모달, 날짜 선택 등 반복 사용되는 화면 요소의 사용성을 개선했습니다.
