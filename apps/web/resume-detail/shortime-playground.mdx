## 기본 정보

- 회사: 주식회사 코드크레용
- 기간: 2024.09 - 2025.04
- 고용 형태: 정규직
- 핵심 포지션: 앱 배포와 독립된 모바일 웹뷰 제품
- 기술 스택: Next.js, React, TypeScript, Tailwind CSS, Phaser, Next Intl, Flutter InAppWebView, Web Audio API, Airbridge, Branch.io

## 핵심 요약

앱 배포와 별개로 이벤트와 게임을 업데이트할 수 있는 Playground를 초기 설계부터 개발했습니다.

## 문제

- 앱 배포와 독립적으로 이벤트·운세·게임·드로우 콘텐츠를 추가할 구조가 필요했습니다.
- Flutter WebView 통신과 딥링크·광고를 연결하고, 여러 모바일 브라우저의 동작 차이에 대응해야 했습니다.

## 해결·결과

- 이벤트·운세·게임·드로우를 도메인 단위로 나눠 앱 배포와 독립적으로 콘텐츠를 추가할 수 있게 했습니다.
- Flutter WebView 통신, 딥링크와 리워드 광고를 연결하고, 게임 결과 제출 중 중복 입력과 화면 상태를 제어했습니다.
- Chrome, Safari와 Samsung Internet의 키보드·포커스·화면 안전 영역·테마·오디오 차이에 대응했습니다.
