## 기본 정보

- 회사: 주식회사 코드크레용
- 기간: 2025.02 - 2025.07
- 고용 형태: 정규직
- 핵심 포지션: AI 자막 번역과 콘텐츠 운영 자동화
- 기술 스택: Node.js, TypeScript, Express, AWS Bedrock, OpenAI API, Google Cloud Translation, Axios, Cheerio, Vercel Serverless Functions

## 핵심 요약

반복 운영 업무를 운영자가 계속 사용하는 웹 도구로 전환했습니다.

## 문제

- 자막 번역이 CLI와 개발자 중심 흐름에 머물러 운영자가 직접 다루기 어려웠습니다.
- 무단 도용 콘텐츠 확인도 브라우저 검색에 의존해 매일 반복 시간이 들었습니다.

## 해결·결과

- 자막 번역 CLI를 웹 관리 도구로 전환해 운영자가 파일·프롬프트·검수·캐릭터 매칭을 처리하게 했습니다.
- 청크·병렬 번역, 실패 재요청과 자막 수·누락·시간 검증을 적용했습니다.
- 도입 전 3명이 하루 1~2개 처리하던 번역을 도입 후 1명이 하루 6~8개 처리할 수 있게 했으며, 해당 시스템은 현재도 사용되고 있습니다.
- 브라우저에서 차단되던 검색을 Node.js 서버로 옮기고, 3개 플랫폼의 검색과 유사 제목 필터링을 자동화해 하루 50분을 10분 이내로 줄였습니다.
