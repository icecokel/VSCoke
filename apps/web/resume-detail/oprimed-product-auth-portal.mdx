## 개요 (Overview)

여러 제품으로 이동하는 공통 로그인 화면과 세션 만료 경험을 개선했습니다.

## 기술 스택

Next.js · React · TypeScript · Playwright · GitHub Actions

## 주요 작업

- 제품마다 로그인, 세션 만료, 이동 방식이 달라지는 문제를 공통 진입점과 인증 흐름 기준으로 정리했습니다.
- 여러 제품으로 이동하는 제품 카드 화면과 로그인 이후 이동 흐름을 개선했습니다.
- 세션 만료 시 공통 모달을 노출하도록 만들어 인증 실패 경험을 통일했습니다.
- 모바일 접근 안내와 이전 접속 경로 기반 로그인 이동 흐름을 보정했습니다.
