## 개요 (Overview)

환자 정보, 질병, 약물, 검사값, 분석 조건을 입력하고 결과를 확인하는 의료·임상 분석 화면을 개발했습니다.

## 기술 스택

Next.js · React · TypeScript · TanStack Query · Zustand · React Hook Form · ECharts · Plotly

## 주요 작업

- 복잡한 분석 조건을 입력, 검증, 실행, 결과 확인 단계로 나누어 화면 흐름을 개발했습니다.
- 서버 기능 명세 변경에 맞춰 요청·응답 타입과 화면 상태를 정리했습니다.
- 저장된 분석 조건과 엑셀 데이터를 다시 불러오는 흐름을 보정했습니다.
- 분석 결과를 테이블, 차트 확대 보기, 문서 미리보기·다운로드로 확인할 수 있도록 구성했습니다.
