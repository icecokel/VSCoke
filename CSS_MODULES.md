# 모듈 CSS를 사용하는 컴포넌트 목록

이 프로젝트에서 CSS Modules를 사용하는 컴포넌트들의 목록입니다.

## Base UI Components

### 1. tooltip
- 파일: `src/components/base-ui/tooltip/style.module.css`
- 컴포넌트: `src/components/base-ui/tooltip/index.tsx`
- 사용 클래스: `.wrapper`, `.text`

### 2. slide
- 파일: `src/components/base-ui/slide/style.module.css`
- 컴포넌트: `src/components/base-ui/slide/index.tsx`

### 3. chip
- 파일: `src/components/base-ui/chip/style.module.css`
- 컴포넌트: `src/components/base-ui/chip/index.tsx`

### 4. dialog
- 파일: `src/components/base-ui/dialog/style.module.css`
- 컴포넌트: `src/components/base-ui/dialog/index.tsx`
- 사용 클래스: `.wrapper`, `.icon`

### 5. button
- 파일: `src/components/base-ui/button/style.module.css`
- 컴포넌트: `src/components/base-ui/button/index.tsx`

### 6. menu
- 파일: `src/components/base-ui/menu/style.module.css`

### 7. text

### 8. modal
- 파일: `src/components/base-ui/modal/style.module.css`
- 컴포넌트: `src/components/base-ui/modal/index.tsx`

### 9. snack-bar
- 파일: `src/components/base-ui/snack-bar/style.module.css`

## 총 9개 컴포넌트

모든 CSS Module 파일은 `base-ui` 디렉토리 내의 각 컴포넌트 폴더에 위치하며, 파일명은 `style.module.css`로 통일되어 있습니다.
