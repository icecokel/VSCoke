## 기본 정보

- 회사: 주식회사 코드크레용
- 기간: 2023.05 - 2025.07
- 고용 형태: 정규직
- 핵심 포지션: 운영 자동화, 모바일 웹뷰, 커머스·백오피스 개발
- 기술 스택: Next.js, React, TypeScript, MUI, Tailwind CSS, Jotai, SWR, React Hook Form

## 핵심 요약

초기 멤버로 합류해 커머스, 영상 플랫폼, 모바일 웹뷰, 백오피스, 운영 자동화 도구를 개발했습니다. 고객용 화면과 운영자 업무 화면을 함께 맡았고, 반복되는 콘텐츠 운영 업무를 웹 도구로 옮겼습니다.

## 문제

- 고객용 커머스 화면과 운영자 백오피스를 함께 개발해야 했습니다.
- 주문, 상품, 셀러, 정산, 이벤트, 문의 관리처럼 운영자가 매일 확인하는 업무 화면이 필요했습니다.
- 기존 백오피스는 새 프론트엔드 구조로 옮기며 운영자 작업 흐름을 다시 잡아야 했습니다.

## 해결·결과

- 다국어, 결제, 선물하기, 드로우, 상품·주문·마이페이지 화면을 개발했습니다.
- 기존 백오피스를 새 프론트엔드 구조로 옮기고 주문, 상품, 셀러, 정산, 이벤트, 문의 관리 화면을 개발했습니다.
- 검색 조건 초기화, 파일 내보내기, 드래그 앤 드롭 정렬, 공통 테이블·다이얼로그·폼 컴포넌트를 적용했습니다.
