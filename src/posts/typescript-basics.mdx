# TypeScript 시작하기

TypeScript는 JavaScript에 정적 타입을 추가한 언어입니다.

## 기본 타입

```typescript
const name: string = "홍길동";
const age: number = 25;
const isActive: boolean = true;
```

## 인터페이스

객체의 구조를 정의할 수 있습니다.

```typescript
interface User {
  name: string;
  age: number;
  email?: string;
}
```

## 제네릭

재사용 가능한 컴포넌트를 만들 수 있습니다.

```typescript
const identity = <T>(arg: T): T => {
  return arg;
};
```
