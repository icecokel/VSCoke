## 기본 정보

- 회사: 주식회사 코드크레용
- 기간: 2023.05 - 2025.07
- 고용 형태: 정규직
- 핵심 포지션: 고객용 서비스와 운영자 도구 개발
- 기술 스택: Next.js, React, TypeScript, MUI, Tailwind CSS, Jotai, SWR, React Hook Form, i18next, GA4, Google Tag Manager, Google Sheets API

## 핵심 요약

초기 멤버로 고객용 서비스와 운영자 도구를 함께 개발했습니다.

## 문제

- 고객용 다국어 커머스와 운영자 백오피스를 함께 개발해야 했습니다.
- 번역 파일과 운영 데이터의 반복 작업을 줄이고, Flutter Web 백오피스를 전환할 필요가 있었습니다.

## 해결·결과

- 다국어 커머스의 상품·주문·결제·선물·드로우·마이페이지를 개발하고, 조회·찜·가입·구매를 GA4/GTM 이벤트로 연결했습니다.
- 4개 언어의 번역 키를 코드에서 수집해 Google Sheets와 번역 파일을 양방향 동기화했습니다.
- Flutter Web 백오피스를 Next.js로 전환하고 상품·셀러 승인, 주문·클레임, 정산과 문의를 개발했습니다.
- 검색 초기화, CSV·Excel 내보내기, 드래그 정렬과 공통 테이블·다이얼로그를 적용했습니다.
