## 개요 (Overview)

여러 프론트엔드 프로젝트의 CI/CD, 실행 환경, 테스트, 배포 준비 흐름을 정리했습니다.

## 기술 스택

GitHub Actions · Docker · Shell Script · Next.js · Playwright · TypeScript

## 주요 작업

- 프로젝트별 실행 환경, 의존성, 패치 버전을 맞춰 개발 환경 차이를 보정했습니다.
- GitHub Actions 기반 CI/CD 워크플로우와 타입 검사, 빌드 검증, 브라우저 자동 테스트 스크립트를 정리했습니다.
- Docker 패키징, 배포 보조 스크립트, 실행 환경 설정을 정리했습니다.
- 배포 전 확인할 기준을 문서와 스크립트로 정리해 검증 흐름을 보강했습니다.
